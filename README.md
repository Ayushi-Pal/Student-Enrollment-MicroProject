# Student Enrollment Micro Project

## Description
This micro project is a Student Enrollment Form developed as part of academic coursework.
The system allows users to insert and update student records using Roll Number as the
primary key. The project is integrated with JsonPowerDB for backend data storage.

---

## Benefits of using JsonPowerDB
- High performance and low latency database
- Schema-free and easy to use
- REST API based database
- Faster development and execution
- Secure and lightweight
